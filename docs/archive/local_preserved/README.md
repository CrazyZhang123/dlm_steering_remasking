# 本地保留归档说明

本目录用于保存**主工作区本地未提交内容**中与实验记录相关的文件。

设计目的：

- 在不丢失本地现场的前提下，把主目录 `docs/` 收口
- 将这些本地补充材料纳入 git 管理
- 其中会影响方法结论的内容，已经合并进 `docs/experiment_summary.md`

使用规则：

- 这里的文件优先视为“本地保留快照”，不是新的主结论文档
- 若与 `docs/archive/` 下的正式归档文档重复，以：
  - `docs/experiment_summary.md` 为最终摘要
  - `docs/archive/local_preserved/` 为本地补充快照

本轮收口后，本目录主要包含：

- 本地未跟踪的阶段日志与会话笔记
- 从旧主目录移动过来的结果型文档
- 本地修改过的旧归档文档副本
